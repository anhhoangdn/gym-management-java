# gym-management-java

## Project type

This repository is set up as a **plain Java project** (non-Maven, non-Gradle) for Eclipse.

## Expected structure

- `src/` - Java source code (packages: `controller`, `model`, `util`, `view`)
- `bin/` - compiled `.class` output (generated by Eclipse)
- `lib/` - optional external `.jar` dependencies
- `.project`, `.classpath`, `.settings/` - Eclipse metadata

## Import into Eclipse

1. Open Eclipse.
2. Go to **File -> Import -> Existing Projects into Workspace**.
3. Choose the folder `gym-management-java`.
4. Finish import.

## JDK configuration

- This project is configured with **Java 17**.
- In Eclipse: **Project -> Properties -> Java Compiler** and ensure compliance is 17.

## Database notes

- SQL schema dump file: `gymmanagement.sql`.
- Environment placeholders are in `.env`:
	- `DB_URL`
	- `DB_USER`
	- `DB_PASSWORD`

> Current source is still in skeleton stage (many TODOs), so this setup focuses on project structure and import/build consistency.
